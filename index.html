soundsfromthecloud

A tool to download songs from SoundCloud. If you come accross any issues, please report them with an issue.

Functionality

  • Download single track
  • Download any playlist/set
  • Download all of an artists publicly uploaded tracks
  • Download a user's likes

Features

  • Adds song and artist data to the mp3's ID3 tag
  • Grabs the thumbnail of the song from soundcloud and sets as ID3 album art for mp3 file
  • Grabs the description from soundcloud and sets as ID3 embedded lyrics
